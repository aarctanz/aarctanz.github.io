# Personal Portfolio

My personal portfolio website built with Astro + Tailwind CSS.

## 🚀 Tech Stack

- **Astro** - Static site generator
- **Tailwind CSS** - Styling
- **Ioskeley Mono** - Font

## 🛠️ Setup

```sh
npm install
npm run dev
```

## 📦 Build

```sh
npm run build
npm run preview
```

## 📝 Add Blog Posts

Create new `.md` files in `src/content/blog/`:

```markdown
---
title: "Your Post Title"
pubDate: 2024-01-15
description: "Short description"
---

Your content here...
```
